This program was created by Matthew Gotta and Michael Gotta using IntelliJ. This is a java program and should be run using IntelliJ or any other Java IDE.

To run the source code:
1. open IntelliJ and create a new Package in your src folder
2. paste all .java files in this folder into that Package
3. navigate to the "Driver" class
4. The main method is inside this class, you can run the program from here
